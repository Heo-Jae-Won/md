## <span style="color:#802548">_중복 방지 method화 및 로직 변경_</span>

- 원래 처음에 중복 방지를 의도했던 것처럼, 가변 배열을 만들어서 진행하는 방식도 만들었다.
- 맨 처음에는 배열이 채워져 있지 않아 length가 0이다. 
- 그 경우에는 createTempPickedNums()에 들어가서 for문 자체를 돌지 않는다. 
- 따라서 length가 0인 첫번째 경우에만 길이가 1인 int[]을 return한다.

```java
int[] numsArr = new int[6];
int numsIndex = 0;
while (numsIndex < 6) {
	Random random = new Random();
	int number = random.nextInt(45)+ 1;
	int[] nums = createTempPickedNums(numsArr).length == 0 ? new int[1] : createTempPickedNums(numsArr);
	boolean isDuplicatedNumber = true;
	for (int count = 0; count < nums.length; count++) {
		if (nums[count] == number) {
			isDuplicatedNumber = true;
			break;
		} else {
			isDuplicatedNumber = false;
		}
	}
	if(!isDuplicatedNumber) {
		numsArr[numsIndex] = number;
		numsIndex++;
	}
	}
```

- 그렇다면 이제 핵심 역할을 담당하는 method를 살펴보자.
	- int배열은 값이 없는 경우 0으로 초기화되므로 0인 값들은 아직 채워지지 않은 값들이다.
	- 로또는 1부터 시작하므로 0이란 값이 실제로 부여된 값인지, 할당되지 않은 값인지 고민할 필요가 없다.
	- 따라서 0은 아직 뽑지 않은 lotto라는 의미다. 해당 방식으로 count를 센다.
	- 다만 이 방법의 경우, 맨 처음 시작할 때가 문제다. 
	- 맨 처음 시작할 때는 모두 0이라서 안에서 for문이 동작하지 않기 때문이다. 
	- 그러나 여기서 if문을 복잡하게 쓰기보단, 바깥에서 해당 method의 배열의 length가 0인 경우를 처리하자.
	- method를 복잡하게 만들지 않아도 깔끔하게 쓸 수 있다.

```java
 public static int[] createTempPickedNums(int[] alreadyPickedNums) {
	
	int pickedNumIndex = 0;
	for (int i = 0; i < alreadyPickedNums.length; i++) {
		if (alreadyPickedNums[i] != 0) {
			pickedNumIndex++;
		}
	}
	
	// 실제 값을 복사해서 가져옴.
	// 복사는 new + for문으로 하여 깊은 복사로 생성
	int[] newArr = new int[pickedNumIndex]; 
	for (int i = 0; i < alreadyPickedNums.length; i++) {
		if (alreadyPickedNums[i] != 0) {
			newArr[i] = alreadyPickedNums[i];
		}
	}
	
	return newArr;
}
```

- 따라서 아래와 같은 로직이 탄생한 것이다.
- length가 0인 경우, 처음이므로 해당 method가 뱉을 int 배열이 텅 비어있다.
- 그러한 경우를 처리하기 위해, 해당 경우에만 길이가 1인 배열 하나를 만들어준다.


## <span style="color:#802548">_정렬 logic 변수명 refactoring_</span>

- m과 k보다는 outer와 inner로 바꿔주자.


- outer는 회차의 개념이고, inner는 교환횟수의 개념이다.
- 이를 반영해 바꿔주자.

```java
for (int outer = 0; outer < lottoNumsNeedsToBeFilled.length; outer++) {

	for(int inner = outer; inner < lottoNumsNeedsToBeFilled.length; inner++) {
		if (lottoNumsNeedsToBeFilled[outer] > lottoNumsNeedsToBeFilled[inner]) {
		temp = lottoNumsNeedsToBeFilled[outer];
```

```java
      for (int i = 0; i < gameCount; i++) {
         int[] lottoNumsNeedsToBeFilled = new int[6];
         
         int curPickedLottoIndex = 0;
         while (curPickedLottoIndex < 6) {
	        Random random = new Random();
	        int lottoNumber = random.nextInt(45)+ 1;
	        int[] alreadyPickedNums = createTempPickedNums(lottoNumsNeedsToBeFilled).length == 0 ? new int[1] : createTempPickedNums(lottoNumsNeedsToBeFilled);
	        boolean hasDuplicatedNumber = validateDuplicationAndReturnFlag(alreadyPickedNums, lottoNumber);
	        if(!hasDuplicatedNumber) {
	            lottoNumsNeedsToBeFilled[curPickedLottoIndex] = lottoNumber;
	            curPickedLottoIndex++;
	        }
         }
          
	      for (int stepCount = 0; stepCount < lottoNumsNeedsToBeFilled.length; stepCount++) {
	         int temp = 0;
	         for(int exchangeCount = stepCount + 1; exchangeCount < lottoNumsNeedsToBeFilled.length; exchangeCount++) {
		         if (lottoNumsNeedsToBeFilled[stepCount] > lottoNumsNeedsToBeFilled[exchangeCount]) {
		            temp = lottoNumsNeedsToBeFilled[stepCount];
		            lottoNumsNeedsToBeFilled[stepCount] = lottoNumsNeedsToBeFilled[exchangeCount];
		            lottoNumsNeedsToBeFilled[exchangeCount] = temp;
		         }
	         }
	      }
	      System.out.print("[ " + (i + 1) + "번 ] ");
	      for (int s : lottoNumsNeedsToBeFilled) {
	          System.out.print(s + " ");
	      }
	      System.out.println();
      }
      scan.close();
  }
   
   /**
    * 이전의 숫자들을 담고 있는 임시배열을 생성. 중복된 값이 있나 비교하는 용도
    * @param alreadyPickedNums
    * @return
    */
   public static int[] createTempPickedNums(int[] alreadyPickedNums) {
	   int pickedNumIndex = 0;
	   for (int i = 0; i < alreadyPickedNums.length; i++) {
		   if (alreadyPickedNums[i] != 0) {
			   pickedNumIndex++;
		   }
	   }
	   
	   int[] newArr = new int[pickedNumIndex]; 
	   for (int i = 0; i < alreadyPickedNums.length; i++) {
		   if (alreadyPickedNums[i] != 0) {
			   newArr[i] = alreadyPickedNums[i];
		   }
	   }
	   
	   return newArr;
   }
   
   /**
    * 
    * @param alreadyPickedNums
    * @param lottoNumber
    */
   public static boolean validateDuplicationAndReturnFlag(int[] alreadyPickedNums, int lottoNumber) {
	   boolean hasDuplicatedNumber = true;
       for (int loopCount = 0; loopCount < alreadyPickedNums.length; loopCount++) {
	   	if (alreadyPickedNums[loopCount] == lottoNumber) {
	   		hasDuplicatedNumber = true;
	   		break;
	   	} else {
	   		hasDuplicatedNumber = false;
	   	}
       }
       
       return hasDuplicatedNumber;
   }
```

## <span style="color:#802548">_class와 method를 이용해 간략하게 쓰기_</span>

- 위와 똑같은 결과를 산출하지만, 매우 간단하다.
- 각 class의 특성을 이해하는 것이 얼마나 중요한 지 알 수 있다.
- hashset은 unique 성을 보장하므로 중복 방지용도로 쓴다.
- stream은 sort를 알아서 해주므로 sort를 쓴다.

```java
package com.example;

import java.util.HashSet;
import java.util.Random;
import java.util.Scanner;
import java.util.Set;

public class Example6 {
   
   
   public static void main(String[] args) {
      Scanner scan = new Scanner(System.in);

      System.out.print("로또를 몇 판 하실라우?: ");
      int gameCount = scan.nextInt();
      
      for (int i = 0; i < gameCount; i++) {
		//중복방지
         Set<Integer> set = new HashSet<>();
         while (set.size() < 6) {
	        Random random = new Random();
	        Integer lottoNumber = random.nextInt(45)+ 1;
	        
	        set.add(lottoNumber);
         }
          
         
         //오름차순 정렬
         System.out.print("[ " + (i + 1) + "번 ] ");
         set.stream().sorted().forEach(s -> System.out.print(s + " "));
	     System.out.println();
      }
      scan.close();
  }
   

}
```
