## <span style="color:#802548">_past File API io package_</span>

- java의 1.2 버전 File API를 쓰게 되면 io package의 inputStream과 outputStream을 사용하게 된다.

```java
        e.printStackTrace();
```

## <span style="color:#802548">_current File API nio package-- FileChannel_</span>

- 하지만 ChannelInputStream은 I/O package와 nio package의 통합을 위한 중간다리 class다.
- 따라서 blocking i/o의 특성을 갖고 있어 대용량 처리에는 여전히 아쉬운 부분이 많다.
- 따라서 그 경우에는 아예 non-blocking 특성을 지닌 FileChannel class를 활용한다.
- 다만 아래처럼 그냥 쓰기만 해선 blocking i/o의 특성이 유지된다. FileChannel은 기본적으로 blocking api다.

```java
// Using FileChannel for efficient file reading and writing
try (FileChannel fileChannel = FileChannel.open(filePath, StandardOpenOption.READ);
        WritableByteChannel outputChannel = Channels.newChannel(response.getOutputStream())) {
    
    long fileSize = fileChannel.size(); // Get the size of the file
    long position = 0; // Position to start reading from in the file
    long chunkSize = 1024 * 1024; // 1 MB chunk size for reading in chunks

    // Transfer the file in chunks to the response output stream
    while (position < fileSize) {
        long bytesTransferred = fileChannel.transferTo(position, chunkSize, outputChannel);
        position += bytesTransferred; // Update position after transfer
    }

    // No need for explicit flush, it's handled by try-with-resources
} catch (IOException e) {
    e.printStackTrace();
    // Handle any errors during the file read/write process
    return "Error during file download.";
}

// Return null, as the response has already been written
return null; // This will be ignored because response.getOutputStream() is already committed
```

- FileChannel을 non blocking으로 바꾸려면 많이 복잡해진다.
- blockingmode를 false로 놓고, selector와 SelectableChannel를 사용해야 하기 때문이다.

```java
public class NonBlockingFileReader {

    public static void main(String[] args) throws IOException {
        Path filePath = Paths.get("path_to_your_file");

        // Open FileChannel in non-blocking mode
        try (FileChannel fileChannel = FileChannel.open(filePath, StandardOpenOption.READ)) {
            
            // Configure the FileChannel as non-blocking
            fileChannel.configureBlocking(false);
            
            // Create a Selector to manage non-blocking channels
            Selector selector = Selector.open();
            
            // Register the FileChannel with the Selector for read operations
            fileChannel.register(selector, SelectionKey.OP_READ);

            // Create a buffer to hold data
            ByteBuffer buffer = ByteBuffer.allocate(1024);

            while (true) {
                // Select the keys whose channels are ready for I/O operations
                if (selector.select(1000) > 0) {  // timeout set to 1000 ms
                    Set<SelectionKey> selectedKeys = selector.selectedKeys();
                    Iterator<SelectionKey> iterator = selectedKeys.iterator();
                    
                    while (iterator.hasNext()) {
                        SelectionKey key = iterator.next();
                        iterator.remove();
                        
                        if (key.isReadable()) {
                            // Channel is ready to be read
                            FileChannel channel = (FileChannel) key.channel();
                            buffer.clear();  // Clear the buffer before reading
                            
                            // Read from the file channel into the buffer
                            int bytesRead = channel.read(buffer);
                            
                            if (bytesRead == -1) {
                                // End of file reached
                                System.out.println("End of file reached.");
                                return;
                            }

                            buffer.flip();  // Prepare buffer for writing

                            // Process the data (e.g., write to a response output stream)
                            System.out.println("Read " + bytesRead + " bytes");
                            while (buffer.hasRemaining()) {
                                System.out.print((char) buffer.get());  // Example processing
                            }
                        }
                    }
                }
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
```

- 좀 더 간단한 버전은 아래와 같다.
    - 대신 error를 handling하는 CompletionHandler를 설정하지 않는다.
    - 따라서 io가 끝나기도 전에 main thread에서 접근하지 못하게 막아주어야 한다.

```java
import java.nio.channels.*;
import java.nio.file.*;
import java.nio.ByteBuffer;
import java.util.concurrent.*;

@PostMapping("/board/download")
public ResponseEntity<String> download(@RequestParam(name = "boardSeq") Long boardSeq, HttpServletResponse response) {
    BoardDTO boardDTO = boardService.selectOne(boardSeq);

    String savedFileName = boardDTO.getSavedFileName();
    String originalFileName = boardDTO.getOriginalFileName();

    Path filePath = Paths.get("c:/uploadPath1", savedFileName);

    try {
        String tempName = URLEncoder.encode(originalFileName, StandardCharsets.UTF_8.toString());
        response.setHeader("Content-Disposition", "attachment;filename=" + tempName);
    } catch (UnsupportedEncodingException e) {
        e.printStackTrace();
    }

    try (AsynchronousFileChannel fileChannel = AsynchronousFileChannel.open(filePath, StandardOpenOption.READ);
         WritableByteChannel outputChannel = Channels.newChannel(response.getOutputStream())) {

        long fileSize = fileChannel.size();
        ByteBuffer buffer = ByteBuffer.allocate(1024 * 1024);  // 1MB buffer

        // Start the asynchronous file read operation
        Future<Integer> result = fileChannel.read(buffer, 0);
        
        // Wait for the read operation to complete (this is a blocking call for simplicity)
        int bytesRead = result.get();

        while (bytesRead != -1) {
            buffer.flip();
            outputChannel.write(buffer);
            buffer.clear();
            
            // Start reading the next chunk
            result = fileChannel.read(buffer, fileSize - bytesRead);
            bytesRead = result.get();
        }

    } catch (IOException | InterruptedException | ExecutionException e) {
        e.printStackTrace();
        return new ResponseEntity<>(HttpStatus.INTERNAL_SERVER_ERROR);
    }

    return new ResponseEntity<>(HttpStatus.OK);
}
```
